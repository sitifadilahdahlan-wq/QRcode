Konfigurasi Google Apps Script

Secara default, aplikasi berjalan di mode simulasi. Masukkan URL Apps Script Web App Anda untuk mulai menyimpan data langsung ke Google Sheets secara real-time.

Dapatkan URL ini setelah Anda mendeploy proyek Apps Script Anda sebagai Web App.

Panduan Langkah Demi Langkah

  1. Buat Google Spreadsheet baru, beri nama lembar aktif (Sheet) pertama Anda ScanData.
  2. Klik menu Ekstensi > Apps Script.
  3. Hapus semua kode di dalam file Code.gs, lalu salin dan tempel kode berikut:
/* KODE BACKEND GOOGLE APPS SCRIPT (Code.gs) */

function doGet(e) {
  // Jika diakses langsung via browser, tampilkan UI HTML
  return HtmlService.createHtmlOutputFromFile('index')
      .setTitle('QR Scanner Pro')
      .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL)
      .addMetaTag('viewport', 'width=device-width, initial-scale=1');
}

// Handler untuk integrasi Standalone Web App URL (menggunakan POST)
function doPost(e) {
  try {
    var params = JSON.parse(e.postData.contents);
    var qrData = params.qrData;
    var timestamp = params.timestamp || new Date();
    
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("ScanData") || SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.appendRow([timestamp, qrData]);
    
    return ContentService.createTextOutput(JSON.stringify({ "status": "success" }))
                         .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({ "status": "error", "message": error.toString() }))
                         .setMimeType(ContentService.MimeType.JSON);
  }
}

// Handler untuk integrasi Native Apps Script Container (via google.script.run)
function saveScanToSheet(qrData) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("ScanData") || SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    sheet.appendRow([new Date(), qrData]);
    return { success: true };
  } catch (error) {
    return { success: false, error: error.toString() };
  }
}
  1. Jika ingin menggunakan mode Native, buat file HTML baru di editor Apps Script Anda, beri nama index, dan paste seluruh isi file HTML ini ke sana.
  2. Deploy sebagai Web App dengan mengeklik tombol Terapkan (Deploy) > Penerapan Baru.
  3. Pilih jenis Aplikasi Web, atur akses "Siapa saja yang memiliki akun Google" atau "Siapa saja (Anyone)" tergantung preferensi privasi Anda, lalu klik Terapkan.
